# Курс по языкам программирования C и Rust

Этот репозиторий содержит учебные материалы, лабораторные и практические работы по языкам программирования C и Rust.

## Содержание курса

### Часть 1: Язык C
1. **Практическая работа №1** - Установка и настройка среды разработки
2. **Лабораторная работа №1** - Указатели и работа с памятью
3. **Лабораторная работа №2** - Структуры и функции
4. **Лабораторная работа №3** - Объединения и перечисления
5. **Лабораторная работа №4** - Продвинутая работа с функциями
6. **Практическая работа №2** - Многофайловые проекты и системы сборки

### Часть 2: Язык Rust
7. **Лабораторная работа №1** - Основы Rust
8. **Лабораторная работа №2** - Ссылки, Ownership, Traits, Generics
9. **Лабораторная работа №3** - Комплексное приложение (библиотека)

## Требования к выполнению

Каждая задача должна содержать:
- Исходный код на C или Rust
- Комментарии в коде
- Краткое описание решения

## Соглашение об именовании

- Папки: `N-topic-name/` (где N - номер работы)
- Файлы с кодом: `task-name.c` или `task-name.rs`
- Документация: `README.md` в каждой папке
- Задачи: `task-01-description/`, `task-02-description/`
